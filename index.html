Image2pixel Help

Image2pixel lets you create pixel art from any image. From photographs to graphics, you can convert any image to pixel art style with customizable settings for pixel size, color count, and more.

Here you will find help documentation, tutorials, and tips to help you use Image2pixel effectively. If you have questions, check out the Frequently Asked Questions.

To get started quickly, you can print the Quick Reference. It contains keyboard shortcuts and common operations, so it's handy to have nearby.

Basics

General concepts, elements in the window, expected workflow, etc.

Workspace

The Image2pixel workspace is where all your pixel art creation happens. It includes the main canvas area, toolbars, and settings panels.

Workflow

A typical workflow includes opening an image, adjusting pixel size and colors, applying the algorithm, and saving your pixel art.

Interface

The interface has a top toolbar for file operations, a side panel for adjusting settings, and the main canvas area to view your work.

Image Processing

How to open, create, and save your pixel art images.

Open & Create

Open existing images via File → Open or drag and drop. Create new blank canvases via File → New.

Resize

Adjust image dimensions using the resize tool. You can maintain aspect ratio or set custom dimensions.

Save

Save your work as PNG, GIF, or other formats. PNG is recommended for preserving transparency.

Animation

Working with animated GIFs and transparency.

GIF Support

Import and export animated GIFs. Each frame will be pixelated while preserving animation timing.

Transparency

Transparency in your original images is preserved in the final pixel art. Great for sprites and game assets.

Algorithms

Understanding and using the different pixelation algorithms.

Standard Algorithm

The default algorithm creates classic pixel art using uniform grid-based pixels with customizable block size and color count.

SLIC Algorithm

Creates content-aware pixel art with irregular-shaped pixel clusters that better follow image features and boundaries.

Color Palettes

Choose from built-in retro palettes or create custom ones to achieve specific visual styles and aesthetics.

Customization

Personalize your Image2pixel experience.

Themes

Customize the application's appearance with different themes. Choose from Light, Dark, or System mode to match your preference and reduce eye strain during long editing sessions.
