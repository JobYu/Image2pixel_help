快速参考

本快速参考指南提供了 Image2pixel 中最常用操作和键盘快捷键的概述。打印此页面，在使用应用程序时可作为方便的参考。

键盘快捷键

文件操作

操作 Windows / Linux macOS
打开文件 Ctrl+O Cmd+O
保存文件 Ctrl+S Cmd+S
退出应用程序 Alt+F4 Cmd+Q

界面元素

主界面组件

组件 描述
打开按钮 点击从系统中打开图像或 GIF 文件
保存按钮 点击保存当前像素艺术结果
块大小滑块 调整输出图像中像素的大小
颜色滑块 调整输出图像中的颜色数量（2-96）
原始预览 显示原始输入图像
结果预览 显示像素化输出图像
算法下拉菜单 选择像素化算法（标准或 SLIC）

算法

标准算法参数

参数 范围 描述
块大小 1-50 输出中每个像素块的大小
颜色 2-96 输出图像中的最大颜色数量

SLIC 算法参数

参数 范围 描述
超像素大小 8-64 超像素区域的大小
颜色权重 1-40 颜色相似度与空间接近度的权重
迭代次数 1-20 SLIC 算法迭代次数
像素步长 1-8 处理像素之间的步长
颜色 2-96 输出图像中的最大颜色数量

文件格式

支持的输入格式

  • PNG (.png)
  • JPEG/JPG (.jpg, .jpeg)
  • GIF (.gif)
  • BMP (.bmp)
  • WebP (.webp)

支持的输出格式

  • PNG (.png)
  • GIF (.gif) - 用于动画输出

调色板格式

  • Adobe 颜色表 (.act)
  • JASC 调色板 (.pal)
