Steam 版本历史

v1.7.3

  • 新增对 Nano Banana (Gemini 2.5 Flash Image) 模型的支持
  • 移除了其他 Google 模型，专注于 Nano Banana
  • 新增对 Doubao-Seedream-4.0 模型的支持
  • 当 API Key 配置正确时，支持直接从模型列表切换模型
  • 支持导入多个自定义颜色调色板

v1.7.2

  • 新增对 Doubao-Seededit-3.0-i2i 和 Doubao-Seedream-4.0 模型的支持
  • 改进了图像到图像模型，自动检测预览窗口图像
  • 修复了转换 blob URL 时的内容安全策略 (CSP) 问题
  • 增强了 API 响应的错误处理

v1.7.0

  • 新增具有高级像素艺术生成的 Pixelizer 算法
  • 增强了具有水平参数布局的界面
  • 提高了算法性能和响应速度
  • 修复了目标宽度参数功能

v1.6.0

  • 重构了代码结构。
  • 优化了 AI 图像生成 UI 交互。

v1.5.1

  • 增强了 API Key 存储安全性。

v1.5.0

  • 优化了 Standard 和 SLIC 算法中的透明度处理，修复了空洞像素。
  • 移除了 AI 生成按钮上的半透明叠加层。

v1.4.2

  • 修复了 Standard 和 SLIC 算法中半透明像素变为空洞的问题。

v1.4.1

  • 显示原始图像和导出图像的尺寸。

v1.4.0

  • 新增用于使用 Bayer 抖动生成单色像素艺术的 1bit 算法
  • 增强了算法选择，具有三种不同的模式：Standard、SLIC Superpixel 和 1bit
  • 改进了用户界面，针对不同算法优化了参数布局
  • 新增对灰度和高对比度艺术风格的支持

v1.3.3

  • 为非 GIF 图像自动保存额外的 1x1 像素版本，在放大作品的同时提供真实的像素表示。
  • 优化了透明图像

v1.3.2

  • 新增对 GIF 和图像中透明背景的支持
  • 改进了所有像素化算法中对 Alpha 通道的处理
  • 增强了对具有透明度的图像的图像质量保存

v1.3.1

  • 优化了加载进度指示器，仅对需时超过 2 秒的操作显示
  • 通过减少快速操作的 UI 闪烁，改善了用户体验
  • 增强了图像处理操作的性能

v1.3.0

  • 新增了 SLIC 算法，用于基于超像素的像素艺术生成
  • 改进了带有 Toast 通知的错误处理
  • 增强了算法参数管理
  • 为 SLIC 算法添加了全面的文档
  • 更新了许可信息

v1.2.0

  • 新增自定义颜色调色板导入（支持 .act 和 .pal 格式）
  • 改进了颜色调色板管理
  • 增强了用户界面

v1.1.2

  • 新增键盘快捷键（Cmd+O/Ctrl+O 打开，Cmd+S/Ctrl+S 保存）
  • 新增用户指南
  • 改进了版本管理

v1.1.1

  • 改进了带有时间戳格式的文件保存
  • 优化了界面语言支持
  • 修复了 GIF 处理问题

v1.1.0

  • 新增 GIF 动画支持
  • 新增 GIF 到像素艺术的转换
  • 改进了颜色量化算法

v1.0.0

  • 初始发布
  • 基本的图像到像素艺术转换
  • 颜色调色板优化（2-96 色）
