快速參考

本快速參考指南提供了 Image2pixel 中最常用操作和鍵盤快捷鍵的概述。列印此頁面，在使用應用程序時可作為方便的參考。

鍵盤快捷鍵

文件操作

操作 Windows / Linux macOS
打開文件 Ctrl+O Cmd+O
保存文件 Ctrl+S Cmd+S
退出應用程序 Alt+F4 Cmd+Q

界面元素

主界面組件

組件 描述
打開按鈕 點擊從系統中打開圖像或 GIF 文件
保存按鈕 點擊保存當前像素藝術結果
塊大小滑塊 調整輸出圖像中像素的大小
顏色滑塊 調整輸出圖像中的顏色數量（2-96）
原始預覽 顯示原始輸入圖像
結果預覽 顯示像素化輸出圖像
算法下拉菜單 選擇像素化算法（標準或 SLIC）

算法

標準算法參數

參數 範圍 描述
塊大小 1-50 輸出中每個像素塊的大小
顏色 2-96 輸出圖像中的最大顏色數量

SLIC 算法參數

參數 範圍 描述
超像素大小 8-64 超像素區域的大小
顏色權重 1-40 顏色相似度與空間接近度的權重
迭代次數 1-20 SLIC 算法迭代次數
像素步長 1-8 處理像素之間的步長
顏色 2-96 輸出圖像中的最大顏色數量

文件格式

支持的輸入格式

  • PNG (.png)
  • JPEG/JPG (.jpg, .jpeg)
  • GIF (.gif)
  • BMP (.bmp)
  • WebP (.webp)

支持的輸出格式

  • PNG (.png)
  • GIF (.gif) - 用於動畫輸出

調色板格式

  • Adobe 顏色表 (.act)
  • JASC 調色板 (.pal)
