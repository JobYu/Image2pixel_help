Steam 版本歷史

v1.7.3

  • 新增對 Nano Banana (Gemini 2.5 Flash Image) 模型的支持
  • 移除了其他 Google 模型，專注於 Nano Banana
  • 新增對 Doubao-Seedream-4.0 模型的支持
  • 當 API Key 配置正確時，支持直接從模型列表切換模型
  • 支持導入多個自定義顏色調色板

v1.7.2

  • 新增對 Doubao-Seededit-3.0-i2i 和 Doubao-Seedream-4.0 模型的支持
  • 改進了圖像到圖像模型，自動檢測預覽窗口圖像
  • 修復了轉換 blob URL 時的內容安全策略 (CSP) 問題
  • 增強了 API 響應的錯誤處理

v1.7.0

  • 新增具有高級像素藝術生成的 Pixelizer 算法
  • 增強了具有水平參數佈局的用戶界面
  • 提高了算法性能和響應速度
  • 修復了目標寬度參數功能

v1.6.0

  • 重構了代碼結構。
  • 優化了 AI 圖像生成 UI 交互。

v1.5.1

  • 增強了 API Key 存儲安全性。

v1.5.0

  • 優化了 Standard 和 SLIC 算法中的透明度處理，修復了空洞像素。
  • 移除了 AI 生成按鈕上的半透明疊加層。

v1.4.2

  • 修復了 Standard 和 SLIC 算法中半透明像素變為空洞的問題。

v1.4.1

  • 顯示原始圖像和導出圖像的尺寸。

v1.4.0

  • 新增用於使用 Bayer 抖動生成單色像素藝術的 1bit 算法
  • 增強了算法選擇，具有三種不同的模式：Standard、SLIC Superpixel 和 1bit
  • 改進了用戶界面，針對不同算法優化了參數佈局
  • 新增對灰度和高對比度藝術風格的支持

v1.3.3

  • 為非 GIF 圖像自動保存額外的 1x1 像素版本，在放大作品的同時提供真實的像素表示。
  • 優化了透明圖像

v1.3.2

  • 新增對 GIF 和圖像中透明背景的支持
  • 改進了所有像素化算法中對 Alpha 通道的處理
  • 增強了對具有透明度的圖像的圖像質量保存

v1.3.1

  • 優化了加載進度指示器，僅對需時超過 2 秒的操作顯示
  • 通過減少快速操作的 UI 閃爍，改善了用戶體驗
  • 增強了圖像處理操作的性能

v1.3.0

  • 新增了 SLIC 算法，用於基於超像素的像素藝術生成
  • 改進了帶有 Toast 通知的錯誤處理
  • 增強了算法參數管理
  • 為 SLIC 算法添加了全面的文檔
  • 更新了許可信息

v1.2.0

  • 新增自定義顏色調色板導入（支持 .act 和 .pal 格式）
  • 改進了顏色調色板管理
  • 增強了用戶界面

v1.1.2

  • 新增鍵盤快捷鍵（Cmd+O/Ctrl+O 打開，Cmd+S/Ctrl+S 保存）
  • 新增用戶指南
  • 改進了版本管理

v1.1.1

  • 改進了帶有時間戳格式的文件保存
  • 優化了界面語言支持
  • 修復了 GIF 處理問題

v1.1.0

  • 新增 GIF 動畫支持
  • 新增 GIF 到像素藝術的轉換
  • 改進了顏色量化算法

v1.0.0

  • 初始發布
  • 基本的圖像到像素藝術轉換
  • 顏色調色板優化（2-96 色）
