版本歷史

v1.3.2

  • 添加了對 GIF 和圖像中透明背景的支持
  • 改進了所有像素化算法中對 Alpha 通道的處理
  • 增強了對具有透明度的圖像的圖像質量保存

v1.3.1

  • 優化了加載進度指示器，僅對需時超過 2 秒的操作顯示
  • 通過減少快速操作的 UI 閃爍，改善了用戶體驗
  • 增強了圖像處理操作的性能

v1.3.0

  • 添加了 SLIC（簡單線性迭代聚類）算法，用於基於超像素的像素藝術生成
  • 改進了帶有 Toast 通知的錯誤處理
  • 增強了算法參數管理
  • 為 SLIC 算法添加了全面的文檔
  • 更新了許可信息

v1.2.0

  • 添加了自定義顏色調色板導入（支持 .act 和 .pal 格式）
  • 改進了顏色調色板管理
  • 增強了用戶界面

v1.1.2

  • 添加了鍵盤快捷鍵（Cmd+O/Ctrl+O 打開，Cmd+S/Ctrl+S 保存）
  • 添加了用戶指南
  • 改進了版本管理

v1.1.1

  • 改進了帶有時間戳格式的文件保存
  • 添加了 CJK 字體支持
  • 優化了界面語言支持
  • 修復了 GIF 處理問題

v1.1.0

  • 添加了 GIF 動畫支持
  • 添加了 GIF 到像素藝術的轉換
  • 改進了顏色量化算法

v1.0.0

  • 初始發布
  • 基本的圖像到像素藝術轉換
  • 顏色調色板優化
